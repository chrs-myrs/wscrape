# wscrape Specification

## Summary

The engine — a CLI for web search, scraping, crawling, and content extraction. No API keys.

## Ownership

Part of the **wscrape** toolkit. This is the canonical spec for the
`wscrape` skill in your fork — edit it to match any customisations you make.

## Customising

The behaviour lives in `skills/wscrape/SKILL.md` and `tools/wscrape`. Keep this spec in sync
with what you change (WHAT it does, not HOW).
